# Tian Jing Portfolio 2026

这是一个用于在线展示工业设计作品的静态作品集网站项目。

该网站采用轻量级前端结构和基于图片的项目展示方式，旨在便于长期维护和内容更新。它适用于部署在 GitHub Pages 上，并可随着时间的推移不断优化项目视觉效果、文本和页面布局。

A static portfolio website project for showcasing industrial design work online.

This site is built for long-term maintenance and easy content updates, with a lightweight front-end structure and image-based project presentation. It is suitable for deployment on GitHub Pages and for continuing to refine project visuals, text, and page organization over time.
